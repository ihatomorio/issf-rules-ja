## おことわり

この日本語訳は参考のために作成されたものであり、日本クレー射撃協会の発行するルールブックを転記したものではありません。

実際の日本国内で実施される射撃大会についてのルールは、大会の責任者などにご確認ください。

この翻訳は日本国内の銃刀法や火薬類取締法の内容に適していない場合があります。

また、この日本語訳について、作者はいかなる保証もいたしません。
